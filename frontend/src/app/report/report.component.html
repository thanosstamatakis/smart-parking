Technologies Used

This has been our university project for a more efficient way of finding a parking spot with emperical data. The main theme in this itteration of the project has been modularity and scalability. This has been achieved through the use of Flask (a Python microframework) and Angular 7 (frontend JS framework), giving us the oportunity to work seperately from one and other by communicating through RESTful apis. To get more information about this project or to run it localy on your machine please visit our GitHub repository page. You are free to download and run the project locally with the instructions from the read-me file.
